# CTF Writeups & Security Research

A curated collection of CTF write-ups and security analyses.

## Overview
This repository documents my approach to solving security challenges with
an emphasis on **vulnerability analysis**, **attack reasoning**, and
**implementation clarity**.

## Write-up Structure
Each write-up follows a consistent format:
1. Overview  
2. Problem Abstraction  
3. Vulnerability Analysis  
4. Attack Strategy  
5. Implementation  
6. Result  
7. Lessons Learned  

## Highlights
- Focus on *why vulnerabilities exist*, not just how to exploit them
- Minimal but clear Python implementations
- Strong emphasis on real-world security implications

## Usage
All write-ups are written in Markdown and can be viewed directly on GitHub
or rendered on a static portfolio website.

## Disclaimer
All content is for educational purposes only.
